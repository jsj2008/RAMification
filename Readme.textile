h1. RAMification

A Tool to manage RAM disks.
This is a small pet project and in no way finished. I'm working on it on a very irregular basis and the current code if far from complete.

* Create RAM Disks
* Remove RAM Disks
* Create Favourites
** Automount _planned_
** Autosync _planned_
